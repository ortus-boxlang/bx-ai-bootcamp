## 📋 Resumen del Bootcamp

| | |
|---|---|
| **Duración** | 8-9 horas (1 día completo) |
| **Nivel** | Principiante Completo |
| **Prerrequisitos** | Conocimientos básicos de programación |
| **Objetivo** | Construir aplicaciones con IA usando BoxLang |

### Lo que Aprenderás

Al final de este bootcamp, podrás:

- ✅ Entender cómo funcionan la IA y los Modelos de Lenguaje Grande (LLM)
- ✅ Hacer tu primera llamada a la API de IA en minutos
- ✅ Construir conversaciones con múltiples turnos y contexto
- ✅ Cambiar entre proveedores de IA (OpenAI, Claude, Ollama)
- ✅ Extraer datos estructurados de las respuestas de IA
- ✅ Crear herramientas que la IA puede usar
- ✅ Construir un agente de IA autónomo
- ✅ Cargar y procesar documentos para sistemas RAG
- ✅ Crear servidores y clientes MCP para distribución de herramientas
- ✅ Agregar voz — texto a voz, transcripción y traducción de audio

### Lo que Construirás

A lo largo de este bootcamp, crearás:

1. 🎱 **Bola Mágica 8** - Tu primera aplicación de IA
2. 💬 **Asistente de Chat** - Bot de conversación con múltiples turnos
3. 🔄 **Selector de Proveedores** - Trabaja con diferentes modelos de IA
4. 📝 **Extractor de Datos** - Salida estructurada con tipos seguros
5. 🌤️ **Bot del Clima** - IA con herramientas en tiempo real
6. 🤖 **Agente Inteligente** - Asistente de IA autónomo
7. 📚 **Sistema RAG** - Soporte al cliente basado en documentos
8. 🌐 **Servidor MCP** - Servicios de IA distribuidos

## 🎯 Prerrequisitos

Antes de comenzar, asegúrate de tener:

### Requerido

1. **BoxLang Instalado** ([Descargar](https://boxlang.io))
   ```bash
   boxlang --version
   # Debería mostrar: BoxLang 1.x.x
   ```

2. **Editor de Texto** (VS Code recomendado)

3. **Al menos UNA clave de API** (elige una):
   - **OpenAI** (Recomendado): https://platform.openai.com/api-keys
   - **Claude**: https://console.anthropic.com/
   - **Ollama** (Gratis/Local): https://ollama.ai

### Opcional pero Útil

- Conocimientos básicos de programación (variables, funciones, bucles)
- Familiaridad con JSON
- Conceptos básicos de línea de comandos

## ⚡ Inicio Rápido

### Configuración en 5 Minutos

1. **Instala el módulo bx-ai**:
   ```bash
   install-bx-module bx-ai
   ```

2. **Configura tu clave de API**:
   ```bash
   # Crear archivo .env
   echo "OPENAI_API_KEY=sk-tu-clave-aqui" > .env
   ```

3. **Ejecuta tu primera llamada a IA**:
   ```java
   // hola-ai.bxs
   respuesta = aiChat( "¡Hola, IA!" )
   println( respuesta )
   ```
   ```bash
   boxlang hola-ai.bxs
   ```

4. **¡Listo!** 🎉

  │   Tú    │───▶│ BoxLang │───▶│   IA    │───▶│Respuesta│
  │(Usuario)│    │   App   │    │Proveedor│    │ (Texto) │
  └─────────┘    └─────────┘    └─────────┘    └─────────┘

   "¿Qué es          aiChat()     API Call      "BoxLang es

  │                     TÉRMINOS CLAVE                           │
  ├──────────────────┬──────────────────────────────────────────┤
  │ LLM              │ Modelo de Lenguaje Grande - IA que       │
  │                  │ entiende y genera texto                  │
  ├──────────────────┼──────────────────────────────────────────┤
  │ Token            │ Una pieza de texto que la IA procesa     │
  │                  │ (~4 caracteres = 1 token)                │
  ├──────────────────┼──────────────────────────────────────────┤

  ├──────────────────┼──────────────────────────────────────────┤
  │ Herramienta      │ Una función que la IA puede llamar       │
  ├──────────────────┼──────────────────────────────────────────┤

## 💡 Consejos para el Éxito

### Antes de Comenzar

1. **Ten tu clave de API lista** - La necesitarás inmediatamente
2. **Ten la documentación cerca** - El [readme principal](../../readme.md) es una gran referencia
3. **Sigue el proceso** - Escribe el código, no solo copies/pegues
4. **Experimenta** - ¡Cambia cosas y ve qué pasa!

### Durante el Bootcamp

1. **Completa todos los laboratorios** - Refuerzan los conceptos
2. **Toma descansos** - Después de las lecciones 3 y 5 son buenos momentos
3. **Haz preguntas** - Si algo no está claro, investígalo
4. **Construye incrementalmente** - Cada lección se basa en la anterior

### Después del Bootcamp

1. **Explora el curso completo** - El [curso de 12 lecciones](../../course/) profundiza más
2. **Revisa los ejemplos** - La [carpeta de ejemplos](../../examples/) tiene más código
3. **Lee la documentación** - La [documentación](https://ai.ortusbooks.com/) cubre todo
4. **¡Construye algo!** - La mejor manera de aprender es haciendo
